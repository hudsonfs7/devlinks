  • AquaDesign

  • O AquaDesign é uma ferramenta de dimensionamento de redes hidráulicas que utiliza as metodologias de cálculo Hazen-Williams e Universal (Colebrook-White). Ele apresenta uma interface intuitiva, permitindo que os usuários ingressem dados como tipo de fluido, temperatura e pressão, e escolham entre os métodos de cálculo disponíveis. Além disso, o aplicativo possui uma biblioteca abrangente de materiais de tubulação, com diâmetros internos configurados, facilitando a seleção rápida e precisa dos tubos necessários para o projeto. O objetivo principal é oferecer uma solução eficiente para o dimensionamento de redes hidráulicas, contribuindo para projetos mais precisos e sustentáveis.
